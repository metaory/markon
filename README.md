## Features

- **Clipboard**: copy, paste
- **File**: save, load
- **Preview**: resizable split
- **GFM**: GitHub Flavored Markdown
- **Highlight**: 250+ langs, 500+ aliases
- **Theme**: light/dark
- **Spellcheck**: toggle spellcheck
- **Local‑only**

- **Export**: PDF/HTML

- **Shortcuts**: command palette, keybinds
- **Theming**: custom CSS hook
